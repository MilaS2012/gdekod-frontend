# ГдеКод — Фронтенд MVP

Платный информационный сервис с базой проверенных промокодов. Подписка 30 ₽/день через мобильного оператора.

---

## Файлы проекта

| Файл | Описание |
|------|----------|
| `promokod-v4.html` | Главная страница / лендинг |
| `gdekod-lk.html` | Личный кабинет пользователя |

Оба файла должны лежать в **одной папке** — они ссылаются друг на друга.

---

## Главная страница (`promokod-v4.html`)

**Что есть:**
- Навигация с категориями: Еда, Одежда, Электроника, Путешествия, Красота
- Каталог промокодов с частичным раскрытием кода (3 символа скрыты)
- Попап верификации — две вкладки: **Войти** и **Подключить**
  - Вход: email/телефон + пароль + восстановление пароля
  - Регистрация: номер телефона → SMS-код → подписка
- Блок «Один шаг до скидки» с описанием сервиса
- Раздел с описанием Личного кабинета
- После входа/регистрации — редирект на `gdekod-lk.html`

**Операторы:** Мегафон, Теле2, Билайн

---

## Личный кабинет (`gdekod-lk.html`)

**Вкладки:**
- 🗂 **Все промокоды** — алфавитный каталог партнёров с поиском, клик → промокоды партнёра
- 🎟 **Мои промокоды** — актуальные коды + история раскрытых с кнопкой «Код не сработал»
- ❤️ **Избранное** — сохранённые промокоды

**Боковая панель:**
- Профиль: телефон, email, статус подписки
- 🧾 Мои чеки — по месяцам → по дням (дата, сумма, фискальный признак, №ККТ, ссылка ОФД)
- 🔑 Смена пароля
- 💬 Поддержка — документы, FAQ, форма обратной связи
- 🚫 Отключить сервис
- Подписка — тариф, оператор, метод оплаты (оператор / карта / СБП через CloudPayments)

## Деплой на Azure Static Web Apps

1. Залить репозиторий на GitHub
2. Azure Portal → Create resource → Static Web App
3. Подключить GitHub репозиторий
4. Build settings:
   - App location: `/`
   - Output location: `/`
   - Build command: *(пусто — статические файлы)*
5. Подключить домен `gde-code.ru` в разделе Custom domains

После настройки каждый push в `main` → автоматический деплой.
