# ГдеКод — проверенные промокоды

Сервис платного доступа к базе актуальных промокодов с датой последней проверки.

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` | Лендинг — главная страница |
| `gdekod-lk.html` | Личный кабинет |

Оба файла адаптированы для десктопа и мобильных устройств (iOS / Android).

- Хостинг: Azure Static Web Apps
